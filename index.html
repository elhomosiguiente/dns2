dns2

Better DNS, using Ethereum.

Download .zip Download .tar.gz View on GitHub

Decentralised, 0 downtime, free domain name system

dns2 exists in the form of a smart contract on the Ethereum blockchain. This means that it is trustless, can't be censored and has no downtime. See the Github directory for the source code, address and JSON interface.

Work that needs doing

The contract needs testing. Also, there aren't any browsers that support dns2 at the moment. Perhaps this could be solved with plugins?
